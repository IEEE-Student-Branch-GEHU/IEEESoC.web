# Security Policy

## Supported Versions

Security updates are actively provided for the following versions of the IEEESoc Web App Portal:

| Version | Supported |
| :--- | :--- |
| 1.x | :white_check_mark: Yes |

## Reporting a Vulnerability

We prioritize the security of our fellowship leaderboard, data displays, and portal features. If you discover a vulnerability or issue (such as cross-site scripting, authentication bypass in the dashboard command terminal, or client data leaks), please report it responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please report security issues directly to the IEEE Student Branch at:
📩 **ieee-soc-security@gehu.ac.in**

Please include in your report:
1.  A descriptive explanation of the security issue.
2.  Step-by-step reproduction instructions (with proof-of-concept scripts or actions).
3.  The severity and impact of the vulnerability.

Once submitted:
*   We will respond to acknowledge receipt within 48 hours.
*   We will release a patch and notify you upon resolution.
*   We request that you keep the report confidential until the fix is deployed.
