IEEEsoc-Bot Environment

Real-time Webhook

In production, the custom IEEEsoc-Bot is installed on all project repositories. When a Project Admin merges a Pull Request (PR) labeled with difficulty settings, the bot hits our webhook to instantly award points and calculate rankings.

Automated Bot Rules:

  • • Easy PR (+10 pts): Standard documentation tweaks, basic CSS fixes, or simple unit test additions.
  • • Medium PR (+30 pts): Feature implementation, backend service routing, or complex API integrations.
  • • Hard PR (+60 pts): Core architectural scaling, kernel adjustments, or cryptographic contract updates.
  • • Mentor Bonus: Reviews completed under 24 hours add +10 points to the Mentor's Impact Score.

Use the Bot Simulator control console on the right side of the dashboard to trigger mock PR merges and watch the leaderboards adapt!
